Сервис для скачивания и анализа текстовых файлов через внешнее API.

---

## Описание

Сервис предоставляет веб-интерфейс для скачивания каталога текстовых файлов порциями (по 3-9 файлов за запрос) и последующего анализа их содержимого. Файлы содержат строки из 500 цифр, сервис подсчитывает частоту встречаемости каждой цифры (от 0 до 9) как в общем, так и по каждому файлу отдельно.

---

## Функциональность

| Функция | Описание |
|---------|----------|
| Скачивание файлов | Автоматическое скачивание всех файлов из каталога с учетом ограничений API (не более 3 файлов за запрос) |
| Визуализация прогресса | Отображение статуса скачивания, счетчиков и времени выполнения |
| Управление файлами | Список скачанных файлов с сортировкой по времени, пагинация |
| Выбор файлов | Возможность выбора нескольких файлов для анализа |
| Анализ данных | Подсчет частоты цифр в выбранных файлах с визуализацией |
| Статистика | Общая статистика по всем выбранным файлам и отдельно по каждому |

## Установка и запуск

### Локальный запуск

Требования: Python 3.11 или выше.

# Клонирование репозитория
git clone https://github.com/haermaeusmora/file_analyzer_service.git
cd file_analyzer_service

# Установка зависимостей
pip install -r requirements.txt

# Запуск тестового API (отдельный терминал)
python test_api.py

# Запуск сервиса (отдельный терминал)
uvicorn app.main:app --host 127.0.0.1 --port 8001 --reload

После запуска сервис доступен по адресу: http://localhost:8001

---

# Запуск через Docker

 Сборка и запуск
docker-compose up --build

 Остановка
docker-compose down

# Использование
Веб-интерфейс
Скачивание данных

Нажать кнопку "Скачать данные"

Сервис автоматически получает имена файлов через API

Скачивает файлы порциями (до 3 за запрос)

Отмечает их как скачанные

Процесс продолжается до полного скачивания каталога

Прогресс отображается в реальном времени

Просмотр файлов

Все файлы отображаются в таблице

Сортировка по времени скачивания (новые сверху)

Пагинация для навигации

Выбор файлов для анализа

Отметить файлы галочками

"Выбрать все на странице" - выбор всех файлов на текущей странице

"Выбрать все файлы" - выбор всех скачанных файлов

"Снять выделение" - сброс выбора

Анализ файлов

Нажать "Произвести расчёты" после выбора файлов

Результаты отображаются в двух вкладках:

Общая статистика: частота каждой цифры во всех выбранных файлах

По файлам: частота цифр в каждом файле отдельно

# API Эндпоинты
Метод	Эндпоинт	Описание
GET	/api/files	Список скачанных файлов с пагинацией
POST	/api/files/download	Запуск скачивания всех файлов
POST	/api/files/analyze	Анализ выбранных файлов
GET	/api/status	Статус скачивания
GET	/	Главная страница

---

# Взаимодействие с внешним API
Сервис интегрируется с внешним API, реализующим следующие эндпоинты:

Метод	Эндпоинт	Описание
GET	/api/files/names	Возвращает 3-9 случайных имен файлов. Пустой список означает завершение скачивания.
POST	/api/files/download	Принимает не более 3 имен файлов. Возвращает ZIP-архив с файлами.
POST	/api/files/downloaded	Отмечает файлы как скачанные. После отметки файлы исключаются из выдачи.

Обработка ограничений
При превышении лимита запросов API возвращает:

Код	Описание	Действие сервиса
429	Too Many Requests	Пауза согласно заголовку Retry-After
403	Forbidden (блокировка на 30 минут)	Ожидание разблокировки
